# Algoritmos e Ideias

Este repositório tem como objetivo reunir **algoritmos simples e interessantes**, criados a partir de ideias curiosas que surgem no dia a dia.

A proposta é usar este espaço como um **laboratório de experimentação**, onde conceitos vistos em vídeos, livros ou discussões podem ser transformados em código.

Muitas vezes encontramos explicações interessantes de **matemática, lógica, física ou teoria de jogos**, e a ideia aqui é pegar esses conceitos e tentar **implementá-los na prática através de algoritmos**.

Não é um repositório focado em produção ou projetos complexos.
A intenção é apenas **explorar ideias, testar conceitos e brincar com código**.
